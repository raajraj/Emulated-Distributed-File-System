Explanation Page

The page gives a description of all Application features available to the end users





Application Features



Feature/Command Feature Description Feature Details
mkdir Create a directory in the file system Click on the 'New Folder' button and enter the directory to be created in the file system. The alert message informs the directory creation status to the users.
ls Listing contents of a given directory In File Explorer section, enter the path of the directory and click on 'Search'. The files in that directory can be viewed below.
cat Displays contents of a file In File Explorer section, in the search bar enter the file path to be displayed, and click on 'Open' button. The file contents are displayed in the 'File Content' section below.
rm Remove a file from the file system In File Explorer section, in the search bar enter the file path to be deleted, and click on 'Delete' button.
put Uploading a file to the file system Click on the 'Upload' button and enter the file path, number of partitions, choose a file to upload from your local machine, and click on 'Upload' button. The alert message informs the upload status to the users.
getPartitionLocations(file) Returns the locations of partitions of the file In File Explorer section, in the search bar enter the file path and hit enter. The partition location details can be viewed on the Information section of the webpage.
readPartition(file, partition#) Returns the content of partition # of the specified file In the Information section that displays the partition location details, enter the number of the partition that you need to read. The partition contents are displayed below.
Search - Find the number of deaths per country Allows users to search for the number of deaths per country for the selected disease In the Search section, open 'Find the number of deaths per country' option and choose a disease and country. The 'Result' section displays the number of deaths.
Search - Find countries for the number of cases selected Allows users to search for the countries that have number of cases between the range specified by the user. In the Search section, open 'Find countries for the number of cases selected' option and choose the disease and enter the range in the input boxes provided. The 'Result' section displays the countries that have cases between the range entered by user.
Analysis - Total Deaths per Country Returns the total number of deaths per country due to a specific disease. In the Analysis section, click on the 'Total Deaths per Country' button to get an analysis on total deaths per country. The 'Result' section displays the output to the user.
Analysis - Average Number of Recovered cases Returns the average number of recovered cases. In the Analysis section, click on the 'Average Number of Recovered cases' button to get a detailed analysis. The 'Result' section displays the output to the user.
DSCI 551 Home Page - Firebase & MySQL Home page comprises of Firebase and MySQL buttons to access the specific implementation. Click on 'Firebase' to access Application Features implemented in Firebase. Click on 'MySSQL' to access Application Features implemented in MySQL.
